2. Bug fix: Profit calculation
Problem: costPrice is stored per-pack (e.g. €300 for a case of 12). The old code multiplied pack cost × bottle-expanded quantity, inflating COGS by up to 12×.

Fix: Replace every COGS/profit calculation with:

costPerBottle = costPrice / packSize
cost = costPerBottle × bottleQty

Apply this fix in all of these locations:

server/routes.ts

Customer analytics profit block (inside customer detail route)
Invoice COGS journal entry creation
Credit note COGS journal entry creation
Batch journal recalculation loop
Per-invoice journal regeneration loop
server/storage.ts

Monthly trend profit calculation
getSalesReport — per-invoice profit
getItemSalesReport — per-line COGS: change costPrice × qty to (costPrice / packSize) × qty

6. Items: Stock reorder suggestions panel
server/storage.ts

Add interface method and implementation for getStockSuggestions():

Compute sixtyDaysAgo = today − 60 days
Fetch all active items (id, name, sku, stockQuantity, reorderLevel, packSize, categoryName)
Fetch all non-cancelled invoices of type "invoice" where date >= sixtyDaysAgo
For each invoice line item, convert to bottles: qty = saleUnit === "pack" ? quantity × packSize : quantity
Build salesMap[itemId] = totalBottlesSold
For each item:
avgMonthly = salesMap[id] / 2
Include item if stockQuantity <= reorderLevel OR (avgMonthly > 0 AND stockQuantity < avgMonthly × 2)
suggestedQty = max(reorderLevel − stock, ceil(avgMonthly × 2 − stock))
urgency = "critical" if out of stock or below reorder with sales, "warning" if below reorder without sales, "info" otherwise
Sort: critical → warning → info, ties by ascending stockQuantity
server/routes.ts

Add GET /api/items/stock-suggestions before GET /api/items/:id:

app.get("/api/items/stock-suggestions", async (_req, res) => {
  const suggestions = await storage.getStockSuggestions();
  res.json(suggestions);
});

client/src/pages/items.tsx

Header row: AlertTriangle icon + "Reorder Suggestions" label + count badge + chevron toggle

Collapsible body: a table with columns: Item (name + SKU + category), In Stock, Reorder At, Avg/Month, Suggested Order
